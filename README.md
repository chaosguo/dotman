About
-----

![Screenshot](https://github.com/xuxiaodong/dotman/raw/master/screenshot.png "screenshot")

A simple dotfile manager.

Installation
------------

    git clone git://github.com/xuxiaodong/dotman.git

Usage
-----

    Usage: dotman -a [-s] [-h]
    
    Options:
        -a, --add   add dot files
        -s, --setup setup dot files
        -h, --help  show help
